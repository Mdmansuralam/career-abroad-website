# Career Abroad Website

A modern, responsive website for an international education and career guidance platform. This website helps students explore study opportunities, connect with universities, and get comprehensive guidance for studying and working abroad.

## 🌍 Features

### Core Sections
- **Hero Section**: Eye-catching slider with call-to-action buttons
- **Search Functionality**: Filter programs by study level, field, and country
- **Services**: University selection, visa assistance, placement services
- **Countries**: Showcase of popular study destinations
- **Testimonials**: Success stories from students
- **Footer**: Contact information, newsletter signup, and links

### Technical Features
- ✅ **Fully Responsive Design** - Mobile, tablet, and desktop optimized
- ✅ **Modern UI/UX** - Clean, professional interface with smooth animations
- ✅ **Interactive Components** - Swiper carousels, smooth scrolling, hover effects
- ✅ **Performance Optimized** - Fast loading, CSS animations
- ✅ **Accessibility** - Semantic HTML, proper contrast, keyboard navigation
- ✅ **SEO Friendly** - Meta tags, proper heading hierarchy

## 🚀 Getting Started

### Prerequisites
- Web browser (Chrome, Firefox, Safari, Edge)
- Basic knowledge of HTML, CSS, JavaScript

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/Mdmansuralam/career-abroad-website.git
cd career-abroad-website
```

2. **Open the website**
   - Simply open `index.html` in your web browser
   - Or use a local server:
   ```bash
   python -m http.server 8000
   # Then visit http://localhost:8000
   ```

### Color Palette

- **Secondary Color**: #D4AF37 (Gold)
- **Dark Color**: #1a1a1a (Near Black)

- **Font Family**: Poppins (Google Fonts)

- Uses rem-based spacing system for consistency
- Mobile-first responsive approach

## 📱 Responsive Breakpoints

- **Desktop**: 1024px and above
- **Tablet**: 768px - 1023px
- **Mobile**: Below 768px
